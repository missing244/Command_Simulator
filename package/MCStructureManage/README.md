# CSstruct
CSstruct结构是命令模拟器为导入器设计的**支持缓冲**和**小数据量储存**的格式。

## CSstruct结构数据格式 
每一个CSstruct结构按以下字节顺序储存结构数据  
* 文件头标识：**b"CSstruct"**
* 作者标识：  
  字节串长度： **1字节(无符号)**  
  数据载体： **utf-8字节串数据**  
* 结构体积：  
  x轴长度： **2字节(无符号)**  
  y轴长度： **2字节(无符号)**  
  z轴长度： **2字节(无符号)**
* 完整结构文件列表:  
  列表长度： **2字节(无符号)**  
  数据载体： **1字节(无符号) + utf-8数据**  
  (Tips: 列表长度的意思是需要解析几段连续的数据载体)
